Man-in-the-Middle Attack Example
Step 1
When a victim requests a web page, the request is directed to the attacker's computer.
Step 2
The attacker's computer receives the request and retrieves the real page from the legitimate website.
Step 3
The attacker can alter the legitimate web page and apply transformations to the data.
Step 4
The attacker forwards the requested page to the victim.
